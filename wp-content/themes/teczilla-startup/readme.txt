== Description ==

Teczilla startup is a child theme of teczilla wordpress theme perfect for startup, finance , startup etc.It is a Very creative, fully customizable and multipurpose theme.Any website you want to make with this theme will look beautiful and compaitable.It has built in typography also dynamic typography also has unlimited color optiopns in it.The theme suitable for saas application website, architecture, interior design, startup, startup, business, decoration, business, furniture, home decor, law firm, startup, insurance, app landing page, it solutions, shop ecommerce, marketing, gym, blog, travel startup, construction,  spa salons, industries, Beauty, photography, and many other websites compatible. Service option has icon and image options. Full Width responsive page layout. SEO optimized pages and fast speed loading.Compatible with gutenberg, elemnetor page builder WooCommerce, contact form 7, Jetpack, Google analytics and so many popular premium & free plugins nicely works with teczilla theme. For more theme information, check out Theme demo here at https://www.avadantathemes.com/demo/teczilla-startup/ and theme instructions here at https://www.avadantathemes.com/documentation/teczilla-free-theme/

Set Homepage Setup:-

1. Go to Dashboard >> Pages >> Add New, Create a new page and name it as "Home" then select the template "Teczilla Home Page" and publish it.

2. Go to Dashboard >> Settings >> Reading, Select the option of Static Page, now select the page you Created As the Home page.
